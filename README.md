## 🧩 Comment l'ajouter à ton projet ?

Sur ton téléphone ou PC, ouvre ton dépôt sur GitHub, crée un dossier `pool/` et ajoute ce fichier `README.md`.  

Ou en ligne de commande depuis ton projet UserLAnd (si tu veux t'entraîner) :

```bash
cd ~/miner-node   # ou le dossier de ton projet
mkdir -p pool
nano pool/README.md   # colle le contenu, sauvegarde
git add pool/README.md
git commit -m "Ajout du guide d'installation de la pool"
git push origin main
